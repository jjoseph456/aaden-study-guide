# 📚 Aaden's Summer Bridge Study Guide

**6th-to-7th Grade | St. Thomas, USVI Curriculum Aligned**

A structured study guide balancing essential 6th-grade review with introductory 7th-grade material. Covers Math, Science, and English Language Arts (ELA) with a focus on vocabulary, inference, and critical thinking.

## 📂 Structure

| Folder | Subject | Focus |
|--------|---------|-------|
| `ela/` | English Language Arts | Synonyms, antonyms, context clues, spelling, reading comprehension & hidden meanings |
| `math/` | Mathematics | Fractions, decimals, ratios, integers, equations, proportions |
| `science/` | Science | Cell biology, ecosystems, earth science, chemistry, plate tectonics |

## 🗓️ Study Plan

- **Weeks 1–3:** 6th-Grade Recall — short weekly 10-question quizzes to rebuild confidence
- **Weeks 4–6:** 7th-Grade Sneak Peek — open-book exams with reading passages

## 🎯 How to Use

1. Start with the diagnostic quiz in each subject folder
2. Review answer keys and explanations after each attempt
3. Track progress using the checkboxes in each quiz file
4. Revisit missed questions weekly
