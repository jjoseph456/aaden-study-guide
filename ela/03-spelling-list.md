# ✍️ ELA: Top 20 Tricky 7th-Grade Spelling Words

> **Practice Strategy:** Have an adult read these aloud to test your spelling against common middle-school traps.

| # | Word | Trap / Memory Trick |
|---|------|-------------------|
| 1 | Caribbean | One 'r', double 'b' |
| 2 | Separate | It has "a rat" in the middle: sep-**a-rat**-e |
| 3 | Definitely | No 'a' in this word; it ends in "-itely" |
| 4 | Embarrass | Double 'b' and double 'r' |
| 5 | Necessary | One 'c', double 's' |
| 6 | Environment | Don't forget the silent 'n' in the middle |
| 7 | Recommend | One 'c', double 'm' |
| 8 | Noticeable | Keep the 'e' before adding "-able" |
| 9 | Mischievous | Pronounced mis-chiv-us, NOT mis-chee-vee-us |
| 10 | Occurred | Double 'c' and double 'r' |
| 11 | Believe | "I" before "E" except after "C" |
| 12 | Foreign | Breaks the "I before E" rule! |
| 13 | Language | Watch out for the "ua" vowel blend |
| 14 | Exaggerate | Double 'g' |
| 15 | Independent | Ends in "-ent", NOT "-ant" |
| 16 | Knowledge | Don't forget the silent 'd' before the 'g' |
| 17 | Maintenance | Changes from "maintain" to "main-ten-ance" |
| 18 | Schedule | The "sch" blend can confuse readers |
| 19 | Guarantee | Starts with "gua" |
| 20 | Government | Often misspelled because people don't pronounce the 'n' |

## 📝 Self-Test

Write each word 3 times from memory, then check against the list above.
